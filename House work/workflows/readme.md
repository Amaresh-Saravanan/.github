# 🌉 SignBridge — Sign Language Translator Web App

SignBridge is an interactive, browser-native Sign Language Translator prototype designed to connect deaf individuals and hearing individuals seamlessly. The application leverages high-fidelity layouts, smooth micro-animations, and client-side web technologies (MediaPipe-ready structures, Web Speech API) to offer a complete end-to-end user and admin experience.

## 🚀 Live Interactive Prototype

The prototype contains **10 fully interactive pages** showcasing both the Customer and Admin flows (SPA routing, settings panels, gesture trainers, speech engines, and interactive dashboards).
